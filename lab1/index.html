¿Cuál es la diferencia entre Internet y la World Wide Web?

Internet es la red global que conecta dispositivos y permite la transferencia de datos, mientras que la World Wide Web es una de las aplicaciones que se ejecuta sobre esa red, permitiendo el acceso a páginas web y recursos multimedia.

¿Cuáles son las partes de un URL?

Las partes de una url son el protocolo de red, el subdominio y el dominio.

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

GET: Solicitar la representación de un recurso. Es utilizado para obtener datos del servidor.

HEAD: Solicitar los encabezados de respuesta de un recurso sin recuperar el cuerpo. Es similar a GET, pero sin el contenido.

Post: Enviar datos al servidor para crear o actualizar un recurso. Se utiliza para enviar datos de formularios, cargar archivos, etc.

Put: Reemplazar un recurso en el servidor con los datos proporcionados. Si el recurso no existe, puede crearlo.

Patch: Aplicar modificaciones parciales a un recurso. A diferencia de PUT, que reemplaza el recurso completo, PATCH solo modifica las partes especificadas.

DELETE: Solicitar la eliminación de un recurso específico en el servidor.

¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?

Cuando accedes a una página web a través de un URL en tu navegador, se utiliza el método HTTP GET. Este método solicita una representación de un recurso específico, como una página HTML, y solo recupera datos sin modificar el estado del servidor

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

Es responsabilidad del desarrollador o equipo de desarrollo corregir los errores 404 en un sitio web. Los desarrolladores deben asegurarse de que los enlaces internos funcionen correctamente y redireccionen a las nuevas páginas de destino.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?

Si un usuario se encuentra con un error 500, es una señal de que algo está mal en el servidor o en la lógica de la aplicación, y es deber del desarrollador investigar y solucionar el problema.

¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

HTML5 ofrece una sintaxis más flexible y coherente en comparación con HTML4.
HTML5 se integra con JavaScript y otras tecnologías para brindar una experiencia web más interactiva y dinámica, mientras que HTML4 trabaja en conjunto con CSS para definir la presentación visual de los elementos.

¿Qué componentes de estructura y estilo tiene una tabla?

caption: Provee un título para la tabla.
colgroup: agrupa columnas temáticamente.
col: aplica atributos comunes y estilos a una o más columnas. Sólo puede ser declarado como hijo de colgroup.
tbody: representa a un cuerpo de la tabla.
thead: representa la sección de encabezado de la tabla.
tfoot: representa la sección de pie de la tabla.
tr: representa una fila. Puede ser declarado como hijo de tbody, thead, tfoot o table.
td: representa a una celda regular. Sólo puede ser declarado como hijo de tr.
th: representa a una celda de encabezado. Sólo puede ser declarado como hijo de tr.
