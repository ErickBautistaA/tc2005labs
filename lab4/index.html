¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Java es un lenguaje copilado mientras que javaScript es un lenguaje interpretado. java tiene una sintaxis más estricta y detallada, y requiere la declaración explícita de tipos de datos. Usa clases y objetos de manera formal. Java es estatico mientras que js es dinamico. Ambos lenguajes comparten algunas similitudes en la sintaxis básica, como el uso de {} para bloques de código y ; para finalizar sentencias. Ambos lenguajes soportan la programación orientada a objetos.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

getDate(), getMonth(), getFullYear(), getHours() y getTime()

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

push(), pop(), map(), filter() y reduce()

¿Cómo se declara una variable con alcance local dentro de una función?

Para declarar una variable con alcance local en js se usa la palabra let o const

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

en las negativas esta la colision de nombres, dificultad en el mantenimiento, difusion de datos, entre otros. en los positivos esta el acceso global, es facil de usar, usa encapsulamiento, etc.
